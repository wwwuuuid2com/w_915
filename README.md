<h3>源码简介：</h3>
<p>店滴AI是一个基于人脸识别的会员管理系统。可应用于商场客流统计、门店智能管理和店内监控等。是一款基于 yii2 的，并且集成了 AI 技术、智能硬件设备对接技术的软硬件开源框架。
它采用 yii2+uniapp+swagger 的开发模式，从接口开发到接口文档都非常快捷方便。同时结合 uniapp 一端开发、多端发布的优势，让一套代码实现 AI 技术对接 app/各种小程序、H5 端的发布。<p>
<p>店滴AI特点：
人脸检测定位
人脸属性分析
150 个关键点定位
情绪识别
图片质量控制
在线图片活体检测<p>
